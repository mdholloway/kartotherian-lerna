tilelive.js is a tile server which supports on-the-fly configuration and 
advanced interaction output. It's based off of Mapnik [1] and uses 
node.js [2] and express [3] for web operations.

[1]: http://mapnik.org/
[2]: http://nodejs.org/
[3]: http://expressjs.com/
