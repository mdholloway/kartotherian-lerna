tilelive.js is a tile server which supports on-the-fly configuration and 
advanced interaction output. It's based off of Mapnik [1] and uses 
node.js [2] and express [3] for web operations.

requirements

- https://github.com/mapnik/node-mapnik


usage

tilelive.js aims to have an equivalent interface to TileLive (the original
version written in Python). Thus instructions for TileLive usage should
apply.

[1]: http://mapnik.org/
[2]: http://nodejs.org/
[3]: http://expressjs.com/
